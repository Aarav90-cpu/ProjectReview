Introduction

Apple introduced the Liquid Glass effect during WWDC 2025—a stunning UI effect that makes interface elements appear to be made of curved, refractive glass.

This project is a hands‑on exploration of how to recreate a similar effect on the web using CSS, SVG displacement maps, and physics-based refraction calculations.

Note: The interactive demos in this project are optimized for Google Chrome due to specific SVG filter support (backdrop-filter).

1. Understanding Refraction

Refraction is what happens when light changes direction as it passes from one material to another (like from air into glass). This bending occurs because light travels at different speeds through different materials.

Snell–Descartes Law

n₁ sin(θ₁) = n₂ sin(θ₂)
  • n₁: Refractive index of first medium (Air ≈ 1)
  • θ₁: Angle of incidence
  • n₂: Refractive index of second medium (Glass > 1)
  • θ₂: Angle of refraction

In our simulation, we assume the ambient medium is air (index=1) and our glass has an index of 1.5. To keep calculations simple, we only calculate one refraction event.

2. Creating the Glass Surface

To create the effect, we must define the shape of the glass. We use a mathematical function to determine the "height" (thickness) of the glass at any point.

[Diagram showing a cross-section of a convex lens]

From this height, we calculate the normal vector (perpendicular to the surface) to determine how light hits the glass. Concave surfaces push light rays outward, while convex surfaces keep them inside.

The Displacement Vector Field

We pre-calculate how much a light ray shifts for every point on the glass. We then normalize these vectors so the maximum displacement is 1. This allows us to store the data in an image.

3. The SVG Displacement Map

Browsers can't read physics formulas directly, so we translate our math into an SVG Displacement Map.

This is an image where pixel colors tell the browser how to shift pixels:

  • Red Channel: Shifts pixels Left/Right (X-axis)
  • Green Channel: Shifts pixels Up/Down (Y-axis)

We use the standard SVG filter <feDisplacementMap /> to apply this to our website elements.

4. Real UI Components

By combining our refraction map with a "Specular Highlight" (a shiny edge reflection), we can build real interface elements.

Magnifying Glass

Uses strong refraction to zoom into content.

Switch

Uses a "lip bezel" (convex outside, concave inside).

Music Player

Mimics the Apple Music glass panel style.

Slider

Sides refract the background while the center remains clear.

Conclusion

This project successfully recreates the core of the Liquid Glass effect using standard web technologies. While currently best viewed in Chrome, it demonstrates that complex physics simulations can be run in the browser to create beautiful user interfaces.
